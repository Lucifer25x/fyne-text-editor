# Simple Text Editor

| Created using fyne

### This is a really bad example of text editor. But it is just for learning.

### Functions:
* Open File
* Save File (Save As)
* New File
* Edit File
* Exit (with confirmation)
* Undo
* Redo (with 5 times limit)
* Find
* Replace
